# Model Load Balance
## 目标
实现模型负载均衡功能，支持在多个模型实例之间分配请求，以提高系统的吞吐量和响应速度。比如, 用户可以在调用模型的时候设定偏好, 看是偏好速度还是偏好质量还是成本还是稳定性还是其他指标, 系统会根据用户的偏好在不同的模型实例之间进行负载均衡。


## 模型供应商
增加多种模型供应商的支持，如火山方舟,Google Gemini, Anthropic Claude, Mistral, DeepSeek, 以及其他主流模型供应商。每个供应商的模型实例可以根据用户的需求进行负载均衡。


**配置供应商 → 注册模型 → 按场景指定系统模型**。业务代码只需按场景名获取模型，零配置调用。

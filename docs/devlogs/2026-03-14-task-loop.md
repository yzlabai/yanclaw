## 架构

```
用户 (Dashboard / Channel /task)
  ↓
TaskLoopController (编排器)
  ↓ spawn / resume
AgentSupervisor → 智能体 (Claude Code / Codex / Gemini)
  ↓ process-stopped(completed)
Verifier (验证产出)
  ↓
TerminationPolicy (判定: done / iterate / blocked)
  ├─ done → Deliverer (交付: git PR)
  ├─ iterate → FeedbackFormatter → resume 智能体
  └─ blocked → 通知用户 → /task resume
```

## Code Review 修复记录

### Critical

| 问题 | 修复 |
|------|------|
| `controller.ts` 缺少 `LoopPreset` import | 补充 import |
| `verifier.ts` 超时后 stdout/stderr 流可能挂起 | hard timeout race + `timedOut` 标志 |
| SSE headers 在 `stream()` 回调内设置 | 移到 `stream()` 调用之前 |

### Important

| 问题 | 修复 |
|------|------|
| `waiting_confirm` 未 touch 进程 → 30min 被 stale eviction | `touchTimers` 每 5min touch |
| `process-stopped(completed)` 后 stale check 竞态 | 收到 completed 立即 `touch()` |
| `task-done` 事件重复发射 | 仅在 delivery 后发一次 |
| Channel 通知未接入 (§6) | `onNotify` + `pushNotification()` + `sendToChannel()` |
| `ConfirmationGate` operations/risk 未接入 | TODO 标注 (需配合 permission-request 事件) |

### Suggestions

| 问题 | 修复 |
|------|------|
| Task 无 GC | `pruneStale(maxAgeMs)` 清理超龄任务 |
| `truncateTail` 重复 | 提取到 `utils.ts` 共用 |
| 前端 SSE 无重连刷新 | `onerror` 调用 `refresh()` |
| SpawnDialog `process.cwd()` | 移除，workDir 改为必填 |
| DevDeliverer 缺 LeakDetector | `setLeakScanner()` + diff 扫描 |

| GET | `/dags/:id` | DAG 详情 |

### Channel `/task` 命令

```
/task <preset> <prompt>              启动任务循环
/task dev <prompt> --path=... --verify="..." --max-iterations=5
/task status                         查看所有任务
/task stop <id>                      停止任务
/task approve <id>                   批准确认断点
/task resume <id> [msg]              恢复阻塞任务
```

### 配置 `config.json5`

```json5
{
  agentHub: {
    taskLoop: {
      enabled: true,
      maxIterations: 10,          // 默认最大迭代
      maxDurationMs: 14400000,    // 默认 4h
      defaultConfirmPolicy: {
        stages: ["delivering"],   // 仅交付前确认
        riskThreshold: "none",    // 全自动
      },
      presets: {
        dev: {
          testTimeoutMs: 300000,  // 5min/command
          testSandbox: "none",
        },
      },
    },
  },
}
```
